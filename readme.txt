How it works.
1. have text editor open and in position 3 in taskbar
2. have course smart open and browser in position 2 taskbar
3. navigate to course smart
4 execute ahk script
5. highlight terms in cousrse smart
6. press shift+1,2,3,4,5 depending on length of the term
7. repeat for as many terms in the chapter
8. once done copy all text into clipboard
9. run fixFlashCards.py
10. paste the formated text into .csv
